インストール方法
---

1. [Tampermonkey](https://tampermonkey.net/)をインストールする
1. <https://mohno007.github.io/typetalk-emoreact/TypetalkEmoreact.user.js> を開く
1. Tampermonkeyのインストール画面が出てきたら、「インストール」ボタンを押下する
